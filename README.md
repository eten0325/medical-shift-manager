# 医療シフト管理システム

医療従事者のシフト管理を効率的に行うためのWebアプリケーションです。

## 機能

- シフトの作成と管理
- スタッフの勤務時間管理
- シフト表の表示と編集
- カレンダー表示

## 開発環境のセットアップ

1. リポジトリをクローン
```bash
git clone [repository-url]
cd medical-shift-manager
```

2. 依存関係のインストール
```bash
npm install
```

3. 開発サーバーの起動
```bash
npm start
```

## 使用技術

- React.js
- JavaScript
- HTML/CSS
